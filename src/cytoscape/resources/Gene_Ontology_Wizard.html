Supported File Formats
Cytoscape supports the following file formats:
  • Cytoscape BioDataServer Format
    • .Onto, .syno and .anno files.
    • Manifest file required.
  • GO Annotation and Ontology Format 
    • OBO and Gene Association files.
    • Latest data is available at Open Biomedical Ontologies
