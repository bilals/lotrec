This directory contains PlugIn JAR files which will be included in the official
Cytoscape 2.0 release.  

If you check a new JAR file into this directory, the Ant script will automatically
include it in a release when you type:  "ant release".

Also, please add a brief description in the pluginsContents.txt file in this directory, as this
is also packaged into the release in the plugins directory.
