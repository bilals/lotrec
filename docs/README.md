## What is LoTREC?

LoTREC is an automated theorem prover for modal and description logics using the **tableau method**. It allows you to:

- **Define logics** by specifying connectors and inference rules
- **Test formulas** for satisfiability or validity
- **Visualize proofs** as interactive graph structures
- **Debug step-by-step** with breakpoints on rules
